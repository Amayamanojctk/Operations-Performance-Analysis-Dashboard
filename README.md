# Operations-Performance-Analysis-Dashboard
End-to-end Business Analyst project using Python, Excel, and Power BI
1:-Project Overview

This project demonstrates an end-to-end Business Analyst workflow, starting from raw data cleaning to interactive dashboard creation and business insights. The goal of this project is to analyze sales and operational performance to support data-driven decision-making.

The project covers ETL, analysis, visualization, and documentation, making it suitable for real-world BA roles.
